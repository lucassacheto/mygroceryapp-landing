About

My Grocery App is a solo Dev application created by Lucas Sacheto. He believes in simplifying the grocery list so people can save money during that "quick stop" at the grocery store.

The app is designed for an efficient experience using cutting-edge technology.

Currently, the web app is in beta; being available for mobile in the future. Check our roadmap for more details.
